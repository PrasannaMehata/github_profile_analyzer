# GitHub Profile Analyzer

Analyze any GitHub profile for:

- 📊 Most used programming languages
- ⭐ Total stars & forks
- 🔁 Repo count & profile rating
- 📈 Chart.js visualization

## Tech Stack
- Python (Flask)
- GitHub REST API
- Chart.js
- HTML/CSS/JS

## How to Use ?

1. Clone repo
2. `pip install -r requirements.txt`
3. Run `python app.py`
4. Open `localhost:5000` and analyze!
